# Sorting Algorithms & Execution Visualization

This folder contains a Python implementation focused on array sorting mechanisms, structural item ordering, and graphical execution visualization.

##  Implemented Methodology
* **Dynamic Array Generation:** Generating randomized numeric structural arrays based on dynamic user matrix inputs.
* **Sorting Execution Logic:** Implementing algorithm routines to step-by-step sort vector values into progressive ascending order.
* **Matplotlib Plot Rendering:** Utilizing complex dimensional subplots (`Figure`) to visually map the data distribution scales before and after the sorting pipeline.

##  Technologies Used
* **Python 3**
* **NumPy / Standard Lists:** For memory array alignment and numeric sequence tracking.
* **Matplotlib:** For programmatic plotting arrays and structural figure rendering.
